# fgl-tools

Tools for working with BOCA Friendly Ghost Language (FGL) — the markup language used by BOCA Lemur / Ghostwriter ticket printers (FGL22/24/26/42/44/46).

## Project status

New repo. No code yet.

## Relevant skills

- `/fgl-layout` — author FGL ticket layouts and convert images (PNG/BMP) to FGL `<g#>` hex graphics.
